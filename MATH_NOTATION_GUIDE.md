# Math Notation Guide

## Overview

The notes section now features automatic math notation formatting that makes mathematical expressions look professional and clean. No special syntax is required - math expressions are automatically detected and formatted.

## Features

### ✨ Automatic Detection
- Math expressions are automatically detected as you type
- No special delimiters or syntax needed
- Works with simple and complex expressions

### 🎨 Clean Formatting
- **Proper superscripting**: Powers are automatically superscripted (e.g., x² instead of x^2)
- **Proper subscripting**: Subscripts are automatically subscripted (e.g., x₁ instead of x_1)
- **Square root notation**: Beautiful √ symbol
- **Greek letters**: Proper Greek letter rendering (α, β, γ, etc.)
- **Uniform font**: Matches surrounding text perfectly
- **No colors or highlighting**: Clean, professional appearance

## How It Works

### Detection Logic
The system intelligently identifies math expressions by:
1. Looking for basic arithmetic operators (+, -, *, /)
2. Looking for power notation (^)
3. Looking for subscript notation (_)
4. Looking for square root notation (sqrt())
5. Looking for simple fractions (a/b)
6. Recognizing Greek letter names
7. Recognizing math symbol names
8. Looking for parentheses with math operators

### Formatting Process
1. **Parse**: Identify math patterns in the text
2. **Convert**: Transform to LaTeX format
3. **Render**: Use KaTeX for beautiful mathematical rendering
4. **Style**: Apply uniform styling that matches surrounding text

## Supported Notation

### Basic Arithmetic
Use standard arithmetic operators:
```
2+2, 5-3, 4*2, 10/2
x+y, a-b, p*q, x/y
```

### Powers
Use caret symbol (^) for superscripts:
```
x^2, x^3, y^4, a^5
```

### Subscripts
Use underscore (_) for subscripts:
```
x_1, x_2, y_3, a_4
```

### Square Roots
Use sqrt() notation:
```
sqrt(4), sqrt(9), sqrt(x)
```

### Fractions
Use forward slash (/) for simple fractions:
```
a/b, x/y, p/q
```

### Greek Letters
Type the full name of the Greek letter:
```
alpha, beta, gamma, delta, pi, omega
```

### Math Symbols
Type the name of the symbol:
```
pi, infinity, infty
```

### Parentheses
Use parentheses for grouping:
```
(x+y), (a-b), (2+3)
```

## Examples

### In Notes
Simply type math expressions in your notes:

```
The sum is 2+2 and the product is 5*3.
The equation is x^2 + y^2 = r^2.
The area is A = pi * r^2.
The square root of 4 is sqrt(4).
```

This will automatically render as:
The sum is 2+2 and the product is 5×3.
The equation is x² + y² = r².
The area is A = π * r².
The square root of 4 is √4.

## Tips

1. **No special syntax needed**: Just type the expression normally
2. **Basic arithmetic**: Use +, -, *, / (e.g., 2+2, 5*3)
3. **Powers**: Use ^ for superscripts (e.g., x^2)
4. **Subscripts**: Use _ for subscripts (e.g., x_1)
5. **Square roots**: Use sqrt() notation (e.g., sqrt(4))
6. **Fractions**: Use / for simple fractions (e.g., a/b)
7. **Greek letters**: Type the full name (e.g., alpha, beta, pi)
8. **Math symbols**: Type the name (e.g., pi, infinity)
9. **Parentheses**: Use () for grouping (e.g., (x+y))

## Troubleshooting

### Expression Not Formatting?
- Ensure the expression matches one of the supported patterns
- Check that the expression is not a chemical formula
- Verify the expression is properly formatted

### Wrong Formatting?
- Make sure you're using the correct notation (^ for powers, _ for subscripts)
- Check for typos in Greek letter names
- Ensure the expression is not too complex

### Chemical vs Math
- Chemical formulas (H2O, CO2) are formatted as chemical formulas
- Math expressions (x^2, sqrt(4), 2+2) are formatted as math
- The system automatically distinguishes between them

## Technical Details

- **Parser**: Custom math expression parser
- **Renderer**: KaTeX for LaTeX rendering
- **Styling**: Custom CSS with uniform font
- **Performance**: Memoized for optimal rendering

## Integration with Chemical Formulas

The system intelligently distinguishes between:
- **Chemical formulas**: H2O, CO2, NaCl → H₂O, CO₂, NaCl
- **Math expressions**: x^2, sqrt(4), 2+2 → x², √4, 2+2

Both are automatically detected and formatted appropriately without any special syntax.
