# Math Notation Test

This document demonstrates improved math notation formatting in notes section.

## Basic Arithmetic

### Addition
- Numbers: 2+2, 5+3, 10+7
- Variables: x+y, a+b, p+q
- Mixed: 2+x, 5+a, 10+y

### Subtraction
- Numbers: 5-2, 10-3, 15-7
- Variables: x-y, a-b, p-q
- Mixed: 5-x, 10-a, 15-y

### Multiplication
- Numbers: 2*2, 5*3, 10*7
- Variables: x*y, a*b, p*q
- Mixed: 2*x, 5*a, 10*y

### Division
- Numbers: 4/2, 10/2, 20/4
- Variables: x/y, a/b, p/q
- Mixed: 4/x, 10/a, 20/y

## Simple Powers and Subscripts

- Powers: x^2, x^3, y^4, a^5
- Subscripts: x_1, x_2, y_3, a_4

## Mixed Examples

- Quadratic formula: x = (-b ± sqrt(b^2 - 4ac)) / 2a
- Pythagorean theorem: a^2 + b^2 = c^2
- Area of circle: A = pi * r^2
- Volume of sphere: V = (4/3) * pi * r^3

## Notes

All math expressions are automatically detected and formatted with:
- Proper superscripting for powers (e.g., x² instead of x^2)
- Proper subscripting for subscripts (e.g., x₁ instead of x_1)
- Beautiful square root notation
- Greek letter rendering
- Uniform font that matches surrounding text
- No special highlighting or colors

The formatting is applied automatically - no special syntax needed!

## Tips

1. **No special syntax needed**: Just type the math expression normally
2. **Powers**: Use ^ for superscripts (e.g., x^2)
3. **Subscripts**: Use _ for subscripts (e.g., x_1)
4. **Square roots**: Use sqrt() notation (e.g., sqrt(4))
5. **Fractions**: Use / for simple fractions (e.g., a/b)
6. **Greek letters**: Type name (e.g., alpha, beta, pi)
7. **Math symbols**: Type name (e.g., pi, infinity)
8. **Basic arithmetic**: Use +, -, *, / (e.g., 2+2, 5*3)
